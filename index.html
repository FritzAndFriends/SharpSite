About Sharp Site

Sharp Site is an open-source content management system built with .NET and Blazor technologies. Developed live on Twitch, Sharp Site aims to provide a flexible and powerful platform for creating and managing web content.

If you're interested in learning more about Sharp Site, check out our GitHub repository for documentation, code examples, and community discussions.
